Project Description:

I created my About Me website to display my personal life and what I have learned throughout this semester of Web Development I. This project demonstrates the use of HTML, CSS, and a small amount of JavaScript, all of which are concepts that we have learned and reinforced throughout the semes.
My site is divided into multiple sections. The Bio and Hobbies sections introduce who I am, my interests, and my goals. The Image Gallery displays some of the custom computers I have built over the years. This section, along with others, includes hover effects to make it more interactive for the user.
The Blog section demonstrates my understanding of a specific web development topic, while also showing how to structure longer written content using HTML and CSS. The Contact Form uses Netlify to allow users to submit messages directly through the site, and gives them the option of whether or not they would like a response. I also included both required and optional fields to collect the necessary information.

Navbar:
The navigation bar serves as the main guide for users moving through the website, helping them quickly access different sections without confusion. I made sure to make use of "scroll-behavior: smooth;" to make it so the page scrolls to the desired location instead of just jumping right to that point. I also added a button effect that changed the color of the button that the user is hovering to make it known to the user where exactly they are selecting.
